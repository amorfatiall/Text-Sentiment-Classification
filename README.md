# Text-Sentiment-Classification

【预选赛】2019中国高校计算机大赛——大数据挑战赛

本预选赛要求选手建立文本情感分类模型，选手用训练好的模型对测试集中的文本情感进行预测，判断其情感为「Negative」或者「Positive」。所提交的结果按照指定的评价指标使用在线评测数据进行评测，达到或超过规定的分数线即通过预选赛。

采用tfidf（特征）+svc（模型）进行预测。（svm.py）
